A — Taller rows (36px), name-hashed colors

Always show full name. "Other" category gets a color based on a hash of the command name — so python3 is always purple, node is always teal, etc. Every unique tool has a distinct consistent color.

Biggest nodes. Always shows full name + category label below it (when wide enough). "Other" still gets hashed colors. Most readable at a glance, but takes more vertical space.
